定期开支（固定/循环记账）

在这里维护每月/每年/每周固定发生的开支，打开网页时会自动补录到当天为止的所有记录，不会重复生成。

云同步（GitHub 当数据库）

数据保存在你的 GitHub 仓库 data.json。任何人打开网页都能看到最新数据；只有填了访问令牌（Token）的人才能写入。Token 只存在你本机浏览器，不会上传。

未连接。填写上面的仓库信息后点「保存设置」，网页会从云端读取数据。
获取 Token：GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → 选择仓库 jizhang，Repository permissions 里把 Contents 设为 Read and write。生成后复制粘贴到上面。
注意：这是 public 仓库，任何人都能看到你的记账数据，请勿在备注里写敏感信息。

导入数据
